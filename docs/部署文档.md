# 部署文档

## 一、本地开发

### 1. 启动数据库

```bash
# 方式 A: Docker (推荐)
docker-compose up -d mysql redis

# 方式 B: 本地安装 MySQL 8.0 + Redis 7.0
mysql -u root -p < server/sql/schema.sql
```

### 2. 启动后端

```bash
cd server
cp .env.example .env
# 修改 .env 中的 DB/Redis/支付配置
npm install
npm run dev
```

后端默认监听 `http://localhost:3000`

### 3. 启动小程序

- 打开微信开发者工具
- 导入 `miniprogram-user` 目录（用户端）
- 导入 `miniprogram-merchant` 目录（商家端）
- 修改 `utils/request.js` 中的 `BASE_URL` 为你的后端地址

> 注意：微信开发者工具需要勾选"不校验合法域名"（开发期）

## 三、生产部署

### 1. 后端部署 (Docker)

```bash
cd server
docker build -t snack-referral-api:latest .
docker run -d \
  -p 3000:3000 \
  -e DB_HOST=your_mysql_host \
  -e REDIS_HOST=your_redis_host \
  -e JWT_SECRET=your_production_secret \
  -e WX_APPID=your_wx_appid \
  -e WX_MCHID=your_mchid \
  -e WX_PAY_V3_KEY=your_v3_key \
  --name snack-api \
  snack-referral-api:latest
```

### 2. 数据库迁移

```bash
mysql -h <host> -u <user> -p<pwd> < server/sql/schema.sql
```

### 3. 小程序发布

1. 微信开发者工具 → 上传代码
2. 微信公众平台 → 版本管理 → 提交审核
3. 支付宝开放平台 → 小程序 → 上传并发布

### 4. Nginx 反向代理

```nginx
server {
  listen 80;
  server_name api.yourdomain.com;

  location / {
    proxy_pass http://localhost:3000;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_set_header X-Forwarded-Proto $scheme;
  }

  location /api/order/pay-callback {
    proxy_pass http://localhost:3000;
    proxy_set_header Host $host;
  }
}
```

### 5. HTTPS 证书 (Let's Encrypt)

```bash
certbot --nginx -d api.yourdomain.com
```

## 四、监控告警

### 1. Prometheus + Grafana (推荐)

```yaml
# prometheus.yml
scrape_configs:
  - job_name: 'snack-api'
    static_configs:
      - targets: ['api.yourdomain.com:3000']
    metrics_path: /metrics
```

### 2. 日志

- 错误日志: `server/logs/error.log`
- 完整日志: `server/logs/combined.log`
- 建议接入 ELK / Loki

## 五、容灾

| 风险 | 应对 |
|---|---|
| 微信支付回调失败 | RabbitMQ 重试队列 + 定时对账 |
| 支付并发漏洞 | Redis 分布式锁 + 数据库乐观锁 |
| 数据库主从延迟 | 关键查询走主库，报表走从库 |
| Redis 宕机 | 降级到数据库查询 |
| 系统瓶颈 | 引入 K8s + HPA 自动扩缩容 |
