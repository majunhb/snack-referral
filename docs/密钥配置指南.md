## 📂 密钥文件放置位置

### 1. 微信小程序消息加密私钥

将 `private.wxdc45c83956b489e9.key` 放到：

```
server/keys/wechat/private.wxdc45c83956b489e9.key
```

### 2. 微信支付 V3 对称密钥（APIv3 Key）

在 `server/.env` 中配置：

```env
WX_PAY_V3_KEY=2y8RiHDs3+S3iq9Or+pHyG66ecNaet8skFEVcHXYez8=
```

### 3. 微信支付平台公钥（非对称密钥）

将 `非对称密钥.txt` 内容保存为：

```
server/keys/wechat/wxpay_public_key.pem
```

### 4. 本地 .env 配置

复制 `server/.env.example` 为 `server/.env`，并填入真实值：

```bash
cp server/.env.example server/.env
```

然后编辑 `server/.env`，填入：

```env
# 微信支付 V3（必填）
WX_APPID=wxdc45c83956b489e9
WX_MCHID=你的微信支付商户号
WX_PAY_V3_KEY=2y8RiHDs3+S3iq9Or+pHyG66ecNaet8skFEVcHXYez8=
WX_NOTIFY_URL=https://yourdomain.com/api/order/pay-callback

# 微信消息加密（可选，如需要消息推送）
WX_ENCODING_AES_KEY_PATH=./keys/wechat/private.wxdc45c83956b489e9.key
WX_ENCODING_TOKEN=你的消息推送 Token
```

## 🚀 快速配置脚本

### Windows (PowerShell)

```powershell
# 1. 创建 keys 目录
New-Item -ItemType Directory -Force -Path "server/keys/wechat"

# 2. 放置私钥
Copy-Item "C:/Users/Administrator/Downloads/private.wxdc45c83956b489e9.key" "server/keys/wechat/"

# 3. 放置微信支付公钥
Copy-Item "C:/Users/Administrator/Downloads/非对称密钥.txt" "server/keys/wechat/wxpay_public_key.pem"

# 4. 创建 .env 文件
Copy-Item "server/.env.example" "server/.env"

# 5. 自动填入对称密钥（需要手动编辑其余字段）
Write-Host "请手动编辑 server/.env 填入商户号等配置"
```

### Linux / macOS

```bash
# 1. 创建 keys 目录
mkdir -p server/keys/wechat

# 2. 放置私钥
cp /path/to/private.wxdc45c83956b489e9.key server/keys/wechat/

# 3. 放置微信支付公钥
cp /path/to/非对称密钥.txt server/keys/wechat/wxpay_public_key.pem

# 4. 创建 .env 文件
cp server/.env.example server/.env

# 5. 自动填入对称密钥
sed -i '' 's/WX_PAY_V3_KEY=.*/WX_PAY_V3_KEY=2y8RiHDs3+S3iq9Or+pHyG66ecNaet8skFEVcHXYez8=/' server/.env
sed -i '' 's/WX_APPID=.*/WX_APPID=wxdc45c83956b489e9/' server/.env

echo "请继续编辑 server/.env 填入商户号等配置"
```
