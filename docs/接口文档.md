# API 接口文档

> 基础地址: `http://localhost:3000/api`
> 所有接口（除登录外）需在 Header 中带 `Authorization: Bearer <token>`

## 统一响应格式

```json
{ "code": 0, "msg": "ok", "data": {...} }
```

---

## 1. 用户相关

### POST /user/login
一键登录/注册

**请求体**:
```json
{
  "platform": "wx",
  "openid": "oXXXX...",
  "phone": "13800138000",
  "nickname": "小吃客",
  "avatarUrl": "https://...",
  "referrerId": 12
}
```

**响应**:
```json
{
  "code": 0,
  "data": { "token": "eyJ...", "userId": 100, "status": 0 }
}
```

### GET /user/info
获取当前用户信息

### POST /user/bind-referrer
绑定推荐人

---

## 2. 推荐相关

### GET /referral/generate
生成专属推荐二维码

**响应**:
```json
{ "code": 0, "data": { "qrCode": "data:image/png;base64,...", "referrerId": 12 } }
```

### GET /referral/stats
我的邀请统计

**响应**:
```json
{
  "code": 0,
  "data": {
    "inviteCount": 5,
    "completedCount": 3,
    "pendingCount": 2,
    "conversionRate": "60.0"
  }
}
```

## 5. 商家后台

### POST /admin/login
**请求体**:
```json
{ "username": "admin", "password": "admin123" }
```

### GET /admin/users
**Query**: `page=1&pageSize=20&status=0&keyword=小吃`

### GET /admin/orders
### GET /admin/coupons

### GET /admin/report
**响应**:
```json
{
  "code": 0,
  "data": {
    "totalUsers": 100, "newUsers": 60, "oldUsers": 40,
    "paidOrders": 50, "totalRevenue": 2500,
    "firstOrders": 60, "couponCount": 80,
    "referralCount": 40, "couponGranted": 40,
    "roi": "2.08"
  }
}
```

### GET /admin/config/list
获取所有配置项

### POST /admin/config/update
更新配置

**请求体**:
```json
{ "key": "coupon_free_max_amount", "value": "50" }
```

## 错误码

| 码 | 含义 |
|---|---|
| 0 | 成功 |
| 401 | 未登录/token 失效 |
| 403 | 无权限 |
| 404 | 资源不存在 |
| 429 | 频控 |
| 500 | 服务器错误 |
| 1001 | 业务参数错误 |
| 1301 | 今日发券已达上限 |
| 1305 | 券状态不可用 |
| 1306 | 券已过期 |
| 1401 | 不能推荐自己 |
| 1504 | 订单不存在 |
| 1505 | 订单已支付 |
| 2001 | 账号或密码错误 |
